Política de Privacidade

Aplicativo: Mestre das Figurinhas
Desenvolvedor: Higor Fernandes

Esta Política de Privacidade descreve como o aplicativo "Mestre das Figurinhas" lida com as informações dos usuários. O aplicativo é fornecido de forma gratuita e destina-se ao uso como está.

Coleta e Uso de Dados Pessoais

O aplicativo "Mestre das Figurinhas" não exige criação de conta, login ou senha. Nós não coletamos, armazenamos ou compartilhamos nenhum dado pessoal identificável dos usuários (como nome, e-mail, telefone ou endereço).

Todas as informações inseridas no aplicativo (como o controle de figurinhas faltantes e repetidas) são salvas localmente no próprio dispositivo do usuário. Se o usuário desinstalar o aplicativo, esses dados locais serão apagados permanentemente.

Serviços de Terceiros e Anúncios (Google AdMob)

O aplicativo utiliza o serviço Google AdMob para a exibição de anúncios. Embora o aplicativo em si não colete dados pessoais, o AdMob (fornecido pelo Google) pode coletar e usar dados automaticamente, como o ID de publicidade do dispositivo, endereço IP e dados de uso, para fornecer anúncios relevantes e medir o desempenho das campanhas.

Para entender como o Google coleta e processa esses dados, recomendamos a leitura da Política de Privacidade do Google em: https://policies.google.com/privacy

Permissões do Dispositivo (Bluetooth e Localização)

Para a funcionalidade "Radar de Trocas P2P", o aplicativo solicita permissões de Bluetooth e de Localização do dispositivo. Essas permissões são estritamente necessárias para a tecnologia Nearby Connections do Android localizar outros colecionadores próximos de forma offline. Nenhuma informação de localização do usuário é mapeada, armazenada em banco de dados ou enviada para servidores externos.

O aplicativo também utiliza as funções nativas de compartilhamento para que o próprio usuário decida se deseja enviar suas listas de figurinhas para outros aplicativos (como o WhatsApp).

Mudanças nesta Política de Privacidade

Podemos atualizar nossa Política de Privacidade de tempos em tempos. É aconselhável revisar esta página periodicamente para quaisquer alterações. Notificaremos sobre quaisquer alterações publicando a nova Política de Privacidade nesta página.
